# Wi-Fi-Password-Viewer
A lightweight Windows utility written in Python that retrieves and displays locally saved Wi-Fi networks and their configurations using native system commands.

## Features
* Scans and lists all saved Wi-Fi profiles on the local computer.
* Displays profile data including cleartext security keys.
* Standalone execution ready (No Python runtime installation required).

## How to Run from Source

### Prerequisites
* Windows Operating System
* Python 3.x installed

### Execution
1. Clone or download this repository to your machine.
2. Open your terminal or Command Prompt inside the project directory.
3. Run the application script using:
   ```bash
   python main.py
